# keep

####Introduction
Generate a simulation screenshot of "Keep" app.

####Description
1.  This project is modified from [南邮keep跑步打卡截图修改](https://gitee.com/jimaske/njupt-keep).
2.  Original author: @jimaske([https://gitee.com/jimaske](https://gitee.com/jimaske)); Modified by: @fyhb([https://gitee.com/fyhb](https://gitee.com/fyhb) / [https://github.com/fy-hb](https://github.com/fy-hb)).

####Changes
1.  Added the function of modifying the avatar in the screenshot.
2.  Added the function of using custom background.
3.  The weather change function is added (currently supported: sunny, cloudy, overcast).
4.  When the page is opened, it will automatically generate and fill in kilometers and pace randomly.
5.  The current time is automatically filled when the page is opened.
6.  The interface in the screenshot was reproduced in high definition.
7.  Fine tuned the organization and layout of some characters in the screenshot.
8.  Altered the font of some words in the screenshot.
9.  Altered some interfaces of the webpage.
10.  Rebuilt the page with javascript.
11. Replaced some materials used in the page.

####Instructions
1.  Modify the contents in the input box to change the patterns appeared in the screenshot.
2.  Click "保存图片"(Save the picture) to generate and save the screenshot.
